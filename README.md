# Art History Study App

A simple React study app for art history exam prep.

## Features

- spoiler-free flashcards
- hide/show answer flow
- study by exam category
- quiz, compare, and cram views
- local artwork images
- keyboard shortcuts: `Space`, `←`, `→`, `S`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Data

Canonical artwork data lives in `src/data/artworks.ts`.

The main grouping field is `examCategory`.

## Images

Local images live in `public/images/`.
